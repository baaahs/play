getGadget

abstract fun <T : Gadget> getGadget(name: String, gadget: T): T

Obtain a gadget that can be used to receive input from a user. The gadget will be displayed in the show's UI.

Parameters

name - Symbolic name for this gadget; must be unique within the show.

gadget - The gadget to display.
