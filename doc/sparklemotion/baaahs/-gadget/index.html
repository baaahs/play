Gadget

abstract class Gadget

Base class for user input gadgets.

Gadgets may be requested by a Show:

val sparklinessSlider = showRunner.getGadget(Slider("Sparkliness"))

fun nextFrame() {
  println("Sparkliness is ${sparklinessSlider.value}.")
}
