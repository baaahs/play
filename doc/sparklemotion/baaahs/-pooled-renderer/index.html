PooledRenderer

interface PooledRenderer

If a Shader.Renderer implements PooledRenderer and pixel prerendering is enabled on Pinky, then the drawing cycle will look like this:

  • shader.createRenderer()
  • rendererN*.beginFrame()
  • pooledRenderer.preDraw()
  • rendererN*.draw()
  • rendererN*.endFrame()
